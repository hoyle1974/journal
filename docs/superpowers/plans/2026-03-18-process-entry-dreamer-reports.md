# Process-Entry & Dreamer Debug Reports Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Write a first-person narrative to the Google Doc for both the `process-entry` Cloud Task and the Dreamer, mirroring the existing query debug report.

**Architecture:** `ProcessEntry` collects its results into a new `ProcessEntryReport` struct, which is returned alongside the `LatencyBreakdown`. `handleProcessEntry` fires an async `gdoc.WriteReport` call (same pattern as `handleQuery`). The Dreamer reuses the LLM narrative already generated by `writeDreamNarrative`, returns it from that function, and `RunDreamer` writes it to the Google Doc. Both paths are gated on `cfg.DebugReportEnabled`.

**Tech Stack:** Go, `text/template`, `internal/prompts`, `internal/gdoc`, `internal/agent`, `internal/api`

func BuildProcessEntryReport(data ProcessEntryReportData) (string, error) {

- [ ] **Step 3: Add `GenerateProcessEntryReport` to `internal/agent/report.go`**

```go
// GenerateProcessEntryReport generates a first-person narrative of what happened during process-entry.
// On failure it returns an empty string so callers can degrade gracefully.
func GenerateProcessEntryReport(ctx context.Context, app infra.ToolEnv, r *ProcessEntryReport) string {
    ctx, span := infra.StartSpan(ctx, "agent.process_entry_report")
    defer span.End()

    if r == nil {
        return ""
    }
    prompt, err := prompts.BuildProcessEntryReport(prompts.ProcessEntryReportData{
        Content:        r.Content,
        Source:         r.Source,
        Significance:   r.Significance,
        Domain:         r.Domain,
        FactStored:     r.FactStored,
        TaskCreated:    r.TaskCreated,
        ContextsLinked: r.ContextsLinked,
        Mood:           r.Mood,
        Tags:           r.Tags,
        EntityNames:    r.EntityNames,
    })
    if err != nil {
        infra.LoggerFrom(ctx).Error("process entry report: prompt build failed", "error", err)
        return ""
    }

    resp, err := app.Dispatch(ctx, &infra.LLMRequest{
        Parts:     []*genai.Part{{Text: utils.WrapAsUserData(prompt)}},
        GenConfig: &infra.GenConfig{MaxOutputTokens: 256},
    })
    if err != nil {
        infra.LoggerFrom(ctx).Error("process entry report: LLM call failed", "error", err)
        return ""
    }

    narrative := strings.TrimSpace(infra.ExtractTextFromResponse(resp))
    infra.LoggerFrom(ctx).Debug("process entry report generated", "length", len(narrative))
    return narrative
}
```
